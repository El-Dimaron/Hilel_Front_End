Таблиці:
    <table>: Визначає таблицю.
        <tr>: Визначає рядок у таблиці.
        <th>: Визначає заголовок стовпця.
        <td>: Визначає комірку даних.

Форми:
    <form>: Визначає форму для введення користувачем даних.
        <input>: Створює поле для введення тексту або інших даних.
        <button>: Створює кнопку для відправлення форми.
        <label>: Визначає мітку для поля вводу.

    <!-- Закоментований текст -->

Списки:
    Ненумерований список (<ul> - Unordered List):
        <ul>
            <li>Перший пункт</li>
            <li>Другий пунк</li>
            <li>Третій пункт</li>
        </ul>

    Нумерований список (<ol> - Ordered List):
        <ol>
            <li>Перший пункт</li>
            <li>Другий пунк</li>
            <li>Третій пункт</li>
        </ol>
    
    Описовий список (<dl> - Definition List):
        <dl>
            <dt>Термін 1</dt>
            <dd>Визначення терміну</dd>
            <dt>Термін 2</dt>
            <dd>Визначення терміну 2</dd>
        </dl>

    Кожен пункт списку визначається тегами <li> (для ненумерованих і нумерованих списків).
    Або тегами <dt> і <dd> (для описового списку). У кожному з цих тегів може бути вкладений інший HTML-код.


Теги-орієнтири (header, main, aside, nav, section, article):
    <header> (заголовок):
        Введення або навігаційна частину сторінки.
        Зазвичай містить заголовки, логотип, навігацію або інші елементи, які входять до верхньої частини сторінки.

    <main> (основний вміст):
        Основний вміст сторінки - не повинен містити навігаційних елементів або заголовків.
        Зазвичай використовується лише один раз на сторінці.


    <aside> (бічний контент):
        Вказує на контент, який стоїть окремо від основного вмісту.
        Може містити бічні панелі, рекламу, пошукові форми чи інші елементи.


    <nav> (навігація):
        Визначає блок з навігаційними посиланнями.
        Зазвичай містить список або інші елементи для навігації.


    <section> (розділ):
        Визначає розділ або тематичну групу контенту на сторінці.
        Зазвичай містить заголовок.


    <article> (основний вміст):
        Вказує на незалежну, самостійну статтю, яка може бути використана окремо від інших частин веб-сторінки.

Ці теги-орієнтири допомагають в створенні структурно зручного і семантично правильного HTML-коду, що полегшує розуміння структури веб-сторінки.


Вбудовані стилі елементу (стилізація тексту):
    font-family - шрифт тексту;
    font-size - розмір шрифту (можна вказувати відності або абсолютні одиниці виміру);
    font-style - стилізація шрифту;
    font-weight - накреслення шрифту (звичайний, напівжирний, жирний тощо);
    text-decoration - декоративні лінії на тексті;
    text-align - горизонтальне вирівнювання в блоці батьківського елемента.
